The WRF Solar Brookhaven model is a real atmospheric physics simulation. It models how clouds affect the sunlight that reaches the ground. The case study is a single day in May 2009, when stratocumulus clouds were observed over the ARM Southern Great Plains site in Oklahoma.

Two parameters control droplet behavior in those clouds. Tuning them by hand or by grid search is impractical because each simulation takes about fourteen hours on six cores.

## The surrogate idea

Run a small batch of real WRF evaluations using Latin Hypercube Sampling. Use those points to train a fast neural surrogate, an MLP that maps parameter pairs to a mean squared error against the observations. The surrogate is wrong, but it is wrong cheaply. Use it as the environment for an SAC reinforcement learning agent that explores thousands of parameter combinations in seconds.

When the agent has a confident guess, run a real WRF simulation at that point. The real result joins the training data, the surrogate is refined, the agent gets smarter. Repeat.

## Why this is interesting

The agent never sees the real WRF directly. It sees a neural impression of it. The impression improves every iteration. The combined system uses real compute only where it matters and pretend compute everywhere else.

This is a Capstone project. The work is being extended from a conference paper into a journal submission with the team at the NJIT Center for Big Data.
