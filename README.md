# danijeun.com

Personal site for Daniel Jeun Valenzuela. Built like a newspaper.

Read `CLAUDE.md` before touching anything. It is the plan, the contract, and the voice guide.

## Quick start

```sh
pnpm install
pnpm dev
```

## Stack

Astro 5, MDX, React 19 islands, Tailwind v4, TypeScript, Biome, Playwright.
Hosted on Vercel. Chatbot retrieval runs on Fly.io.
