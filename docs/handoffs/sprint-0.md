## Decisions baked in
- Output is `static`. The few server functions live as Vercel functions next to the static build.
- Tailwind v4, no `tailwind.config.ts`. Tokens live in CSS via `@theme`.
- Imports use `~/*` alias mapped to `src/*`.
- Prefetching strategy is `viewport`, not `prefetchAll`, to keep payload honest.
